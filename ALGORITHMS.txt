User Registration
Start
  Input username, password, confirm password
  if (length.password >= 6 && confirm password == password && no field is empty)
    store username, password in users table in database
  else
    display appropriate error message
End


User Login
Start
  Input username, password
  Get username and password columns from users table database
  if (username is in (username column) && password is in (password column))
    Print "Login Successful"
    change display (Add data screen)
  else
    Print "Incorrect Username or Password"
End

End
